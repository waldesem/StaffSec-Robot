# Staff-Robot

Staff-Robot an automated bot that parse excel and json files in current directory and insert data to database.

### Usage

```bash
pip install -r requirements.txt
python robot.py
```

### Build and Run Executable

```bash
pip install -r requirements.txt
pyinstaller --clean robot.py
```
